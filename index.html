AI-Based Coding Agent Platform

Quantum ODMR

Designed a low-cost, modular Silicon Carbide-based quantum sensing toolkit using Red Pitaya and RF synthesis. Integrated Arduino-based synthesizer with RF switch and real-time GUI for spectrum visualization. The toolkit includes thermal-controlled laser diode system, clock sync, and lock-in detection with software interface in Python.

Use cases include classroom quantum demos, lab-grade sensing, and field experimentation for magnetometry. System is affordable, reproducible, and tailored for universities and early research labs.
